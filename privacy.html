Privacy Policy for Gmail Bridge App

This application (Gmail Bridge App) is a personal automation tool developed for private use. It accesses Gmail data solely for the purpose of processing emails based on user-defined rules and triggering webhooks.

Data Usage: The application reads email subjects and bodies to apply filtering rules. It does not store, share, or sell any user data. All processing is done in real-time and data is not retained after the webhook is called.

This policy is effective as of June 2025.
